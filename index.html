Acessibilidade é para todo mundo

Este site foi construído com foco em pessoas. Ele demonstra práticas simples que melhoram a experiência de quem usa leitor de tela, navega por teclado ou precisa de mais contraste/tamanho de letra.

  • HTML semântico com landmarks
  • Teclas Tab e Shift+Tab funcionam bem
  • Botão de alto contraste e ajuste de fonte
  • Texto alternativo descritivo nas imagens

Recursos de acessibilidade neste projeto

Navegação por teclado

Ordem de foco lógica, visível e previsível. Elementos clicáveis são acessíveis via teclado.

Alto contraste

Alternância de tema para melhorar a legibilidade em ambientes com pouca luz ou baixa visão.

Ajuste de fonte

Controles para aumentar, diminuir ou resetar o tamanho base de fonte com persistência.

HTML semântico

Uso de header, main, section e footer para orientar tecnologias assistivas.

Galeria descritiva

Veja algumas ilustrações simples em SVG. Cada imagem possui texto alternativo objetivo.
